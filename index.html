Conditional Statements

Conditional Statements are used to execute tasks based on if their conditions have been met. The statements we learned were if, else, else if, and switch; each having their own uses.

  • if: executes if it's specified conditions are met
  • else: executes if the initial if statement does not have it's conditions met
  • else if: changes the flow to check if another statement's conditions are met, if the previous statement's conditions are not met
  • switch: checks multiple statements to see if they are true, and will likely need to use the break command to not return multiple results.
